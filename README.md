# bekoedit

Text editor    
Markdown syncs with editable HTML

Developed as Rust Tauri + Svelte app (with big appreciation)

Designed with in mind:

- Fast speed
- Low memory consumption
- Cross platoform support (Windows, Mac, Linux, variety of BSD)
- Clear and easy UI
- Productivity

## Development

```
bun install
bun run tauri dev
```
